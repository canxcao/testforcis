## Development / Testing

### Install

```sh
$ npm install
```

### Build

```sh
$ npm run build
```

Production build:

```sh
$ NODE_ENV=production npm run build
```

### Test

```sh
$ npm start
```

Open browser and navigate to http://localhost:8080/testforcis

### Watch

```sh
$ npm run watch
```
